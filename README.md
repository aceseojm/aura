# ✦ AURA — AI 운명 상담 서비스

> 사주 + 타로 + AI가 결합한 초개인화 아우라 분석 서비스

## 서비스 철학: Easy-Deep

1. **시작은 쉽고 감각적으로** — 생년월일 먼저 묻지 않기
2. **대화는 길고 깊이 있게** — 최소 3~5회 문답으로 몰입 유도
3. **결과는 시각적이고 전문적으로** — 사주+타로 융합 결과문

## 주요 화면

| 화면 | 설명 |
|------|------|
| 🌙 아우라 싱크 | Canvas 기반 아우라 구체 + 무드 슬라이더 |
| ✦ 운명의 세 갈래 | 타로 / 사주 / 융합(PREMIUM) 선택 |
| 💬 대화형 채팅 | AI 수호신과 멀티턴 대화 + 타로 카드 플립 |
| 💳 결제 완료 | 카카오페이/토스 스타일 영수증 |
| 📄 심층 결과문 | 오행 분포 + 타로 융합 인사이트 + 월별 운세 |

## 기술 스택

- **React 18** (Hooks: useState, useEffect, useRef, useCallback)
- **Framer Motion** — 부드러운 전환 효과
- **Lucide React** — 아이콘
- **Canvas API** — 실시간 아우라 구체 렌더링

## 시작하기

```bash
npm install
npm start
```

## 프로젝트 구조

```
src/
├── App.js                    # 메인 앱 (아우라 오브, 채팅, 페이월)
└── components/
    ├── PaymentSuccess.jsx    # 결제 완료 화면
    └── DeepReport.jsx        # 심층 운명 결과문
```

## 수익화 구조

- 기본 상담: 무료 (타로 or 사주)
- 융합 분석 + 심층 결과문: ₩9,900 (첫 분석 한정가)
- 카카오페이 / 토스 결제 연동
